title: Marking a direction as complete
date: 2026-02-18
---

Users need to be able to complete directions.

## How it works

Users get to the flow by clicking "Complete direction" on the [direction details page](/manage-cases/viewing-direction-details/).

The confirmation page shows the direction description in an inset and asks users to confirm they want to complete it.

![The confirmation page showing "Confirm that you want to complete the direction" as the heading, with an inset showing the direction description, a "Complete direction" button, and a Cancel link.](/manage-cases/marking-a-direction-as-complete/confirm.png)

A cancel link lets users go back to the direction details page without making any changes.

After confirming, the user is taken back to the [case direction list](/manage-cases/viewing-directions-for-a-case/) where a success banner says "Direction completed".

The completed direction no longer appears in the list.

![The case direction list with a green success banner saying "Direction completed" at the top.](/manage-cases/marking-a-direction-as-complete/success.png)

### Activity log

The action is recorded in the activity log as "Direction completed", showing the direction description and a link to view the direction.

![The case activity log showing a "Direction completed" entry by Rachael Harvey, with the direction description and a View direction link.](/manage-cases/marking-a-direction-as-complete/activity-log.png)
